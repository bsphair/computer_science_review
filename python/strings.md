# Python Strings

## Basics

* create string with either " or '

```
computerBrand = 'Apple'

print(computerBrand[0])     # A

print(len(computerBrand))   # 5

```

* string are immutable (can't be changed after they're created)

```
computerBrand = 'Apple'

computerBrand[0] = 'B'    # error
```
